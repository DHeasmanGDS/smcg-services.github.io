Project Examples

ML-Ready Exploration Database (Greenfields Au-Ag)

10 days · Database architecture, QA/QC, automated exports
  • Consolidated drill, geochem, and mapping into a single schema
  • Built validation rules for intervals, duplicates, and assay ranges
  • Delivered “one-click” export templates for filings & analysis

Outcome: Reduced data prep time by ~60%, enabled reproducible ML workflows.

Prospectivity Mapping (Uranium, Basin-hosted)

15 days · Hybrid ML + mineral systems
  • Integrated gravity, magnetics, structure, and proximity features

  • Produced ranked targets with uncertainty bands

Outcome: Short-listed 8 priority targets; informed 2026 field program.

Multi-Parameter Anomaly Detection (Polymetallic)

12 days · Isolation Forests, geochem stacks, validation

  • Delivered reproducible notebooks + dashboard

Outcome: Identified overlooked trend; supported follow-up trenching.

NLP-Driven Knowledge Mining (District Synthesis)

  • Fed hypotheses into prospectivity features

Outcome: Accelerated desktop study; improved feature engineering.
